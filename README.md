# StudyLogTodo-v1.0
1日の勉強においてやるべきことを一元管理できるアプリを制作します！

## 作成目的
2025年夏季から学習を進めてきた C# / WPF / DB の実践として、
「要件定義 → 実装 → 完成」までを冬休みの一週間で経験することを目的に作成しました。

## 主な機能
- タスクの追加、編集、削除
- 完了、未完了の切り替え
- SQLiteによるローカルデータ保存
- 完了済みタスクの一括削除,ダブルクリックで確認後に全削除

## 使用技術
- C#
- WPF (.NET)
- SQLite

## 工夫した点
- App.xamlに共通スタイルをまとめ、
  View側のXAMLが読みやすいように設計しました。
- 一覧画面はスケーラブル、
  追加・編集画面は固定サイズとし、
  操作速度を重視したUIにしています。
- 完成度を負いすぎず、
  機能実装の優先度を考慮しつつ「80点で完成させる」ことを意識しました。

## 今後について
本アプリは v1.0として完成とし、
今後は別の個人制作に取り組む予定です。
